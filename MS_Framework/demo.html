How To Use


Paste Code


Add the following code in the head section of your html file.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Sarthak7500/MyStyle@main/mystyle.css">

<script src="https://cdn.jsdelivr.net/gh/Sarthak7500/MyStyle@main/js/mystyle.js">

Download


Download the latest version of MyStyle Zip File from the link below to your Root Directory and Extract it. And link mystyle.css file in the head section
